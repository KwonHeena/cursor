# 테스트 데이터 생성 가이드

각 교실당 20명씩 총 100명의 학생과 출결 기록을 자동으로 생성하는 스크립트입니다.

## 사용 방법

### 1. 백엔드 디렉토리로 이동

```bash
cd backend
```

### 2. 테스트 데이터 생성 스크립트 실행

```bash
npm run generate-data
```

또는 직접 실행:

```bash
node scripts/generate-test-data.js
```

## 생성되는 데이터

### 학생 데이터
- **총 100명** (교실당 20명씩)
- 각 교실(1~5호실)에 20명씩 배정
- 한국 이름으로 랜덤 생성

### 출결 기록
- 각 학생당 **20~25일**의 출결 기록
- 최근 30일간의 날짜로 랜덤 생성
- 출결 상태 비율:
  - 출석: 70%
  - 결석: 10%
  - 지각: 15%
  - 조퇴: 5%

## 주의사항

⚠️ **기존 데이터 삭제**: 이 스크립트를 실행하면 기존의 모든 학생과 출결 기록이 삭제됩니다!

## 데이터 확인

생성된 데이터는 다음 방법으로 확인할 수 있습니다:

1. **프론트엔드에서 확인**
   - 학생 관리 탭에서 학생 목록 확인
   - 통계 대시보드에서 통계 확인

2. **DBeaver에서 확인**
   - `database/attendance.db` 파일 열기
   - students 테이블과 attendance 테이블 확인

## 데이터 재생성

기존 데이터를 삭제하고 새로 생성하려면 다시 스크립트를 실행하면 됩니다.

```bash
npm run generate-data
```
